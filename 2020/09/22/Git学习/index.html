Git学习

Git本地基础操作

  • git add [file]
  • git commit [-m ‘message’]
  • git rm [file]
  • git diff [file]
  • git reset –hard [commit_id]
    • reset
  • git log
    • 查看提交日志
    • 简化输出添加 --pretty=oneline
    • 查看分支图 --graph
    • 仅显示SHA-1的前几个字符，而非所有的40个字符--abbrev-commit
    • https://www.cnblogs.com/bellkosmos/p/5923439.html git log 命令全解析
  • git reflog
    • 查看命令历史
  • git checkout – [file]

Git远程仓库

  • ssh-keygen -t rsa -C "youremail@example.com"
    • 本地生成SSH Key，用于远程操控免密
  • git remote
    • 远程相关命令 具体可以看提示来使用
    • git remote add origin git@github.com:xxxx@xxx.xxx
  • git push
    • 本地库内容推送到远程
    • -u 参数可以把本地内容推送过去，并且联系两个参赛，自己感受感受,一般都是第一次推送使用。
  • git clone
    • 从远程库克隆

Git分支管理

  • git branch
    • 查看分支
  • git branch [name]
    • 创建分支
  • git checkout [name] / git switch [name]
    • 切换
  • git checkout -b [name] / git switch -c [name]
    • 创建并切换
  • git merge [name]
    • 合并某分支到当前分支
  • git branch -d [name]
    • 删除分支
    • -D强行删除
  • 分支冲突
  • git stash
    • 工作储藏相关命令
    • git stash list
    • git stash apply
    • git stash drop
    • git stash pop
  • git cherry-pick [commit]
    • 把其他提交的命令复制到当前分支，避免重复劳动
  • git pull
    • 从远程库拉取至本地并合并
  • git rebase

git标签管理

  • git tag [name]
    • 给当前分支打上标签
  • git tag
    • 查看所有标签
  • git tag [name] [commit_id]
    • 给commit_id打上标签
  • git show [tag_name]
    • 查看标签信息
  • git tag -d [tag_name]
    • 删除标签
  • git push origin [tag_name]
    • 推送标签到远程
  • git push origin :refs/tags/[tag_name]
    • 删除远程标签

.gitignore文件配置

  • 用的时候在网上找吧

自定义Git

  • git config –global user.name
  • git config –global user.email
